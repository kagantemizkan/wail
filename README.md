## Bilgisayarınızda Çalıştırın


Projeyi klonlayın

```bash
  git clone https://github.com/kagantemizkan/wail
```

Api klasörüne gidin

```bash
  cd api
```
Client klasörüne gidin
```bash
  cd client
```

Gerekli paketleri yükleyin

```bash
  npm install
  npm i sass
```

Sunucuyu çalıştırın

```bash
  npm run start
```

  
## Kullanılan Teknolojiler

**İstemci:** React, SCSS

**Sunucu:** Node, MySQL
